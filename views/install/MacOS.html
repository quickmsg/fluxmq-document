# 安装部署

# 下载安装包

wget https://fluxmq.obs.cn-east-3.myhuaweicloud.com/fluxmq-2.0.12-mac.zip

# License

FluxMQ默认提供100连接的免费接入，安装包下载后自带免费License，如需更多接入数请联系我们获取商业License！

# 启动

JVM参数在fluxmq.sh中，默认10个G堆内存启动

  1. 执行add-executable-permission.sh脚本
sh add-executable-permission.sh
  1. 运行服务
./fluxmq.sh

# 管理页面

版本>2.0.8 FluxMQ会默认包含UI，无需单独安装，其他低版本需要参考管理系统安装

http://ip:8080/
