# 访问控制

访问控制是对发布订阅的ACL的权限控制，例如拒绝某个用户向某个主题发布消息； 主要规则说明：

# 访问控制查询

查询字段 说明
规则类型
1. ALL(所有)
2. IP(IP规则)
3. CLIENTID(客户端ID)
4. USER(用户名)
topic 限制TOPIC
规则 规则类容
动作类型
1. PUBLISH(发布)
2. SUBSCRIBE(订阅)
策略
1. ALLOW(允许)
2. DENY(禁止)
