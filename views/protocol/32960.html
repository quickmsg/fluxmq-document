# GB/T32960

GBT32960жҳҜдёӯеӣҪеӣҪе®¶ж ҮеҮҶгҖҠз”өеҠЁжұҪиҪҰиҝңзЁӢжңҚеҠЎдёҺз®ЎзҗҶи§„иҢғгҖӢзҡ„д»ЈеҸ·гҖӮиҜҘж ҮеҮҶдәҺ2016е№ҙеҸ‘еёғпјҢз”ЁдәҺи§„е®ҡз”өеҠЁжұҪиҪҰзҡ„иҝңзЁӢжңҚеҠЎдёҺз®ЎзҗҶи§„иҢғпјҢеҢ…жӢ¬иҪҰиҫҶиҝңзЁӢиҜҠж–ӯгҖҒж•°жҚ®йҮҮйӣҶгҖҒиҝңзЁӢжҺ§еҲ¶зӯүж–№йқўзҡ„еҶ…е®№гҖӮиҝҷдёӘж ҮеҮҶжңүеҠ©дәҺзЎ®дҝқз”өеҠЁжұҪиҪҰеңЁиҝңзЁӢз®ЎзҗҶе’ҢжңҚеҠЎж–№йқўзҡ„е®үе…ЁжҖ§гҖҒзЁіе®ҡжҖ§е’Ңдә’ж“ҚдҪңжҖ§

еҚҸи®®еҸӮиҖғпјҡ GB/T32960 (opens new window)

# еҗҜеҠЁжңҚеҠЎ

GB/T32960еҚҸи®®жҸ’д»¶еҗҜеҠЁеҗҢж—¶ж”ҜжҢҒ3з§Қи®ӨиҜҒж–№ејҸ,еҰӮжһңй»ҳи®ӨдёҚеҠ еҜҶжҠҘж–ҮеҲҷдёҚйңҖиҰҒдј AES-128гҖҒRSAзҡ„еҜҶй’ҘпјҢе…·дҪ“еҠ еҜҶж–№ејҸз”ұе®ўжҲ·з«ҜеҶіе®ҡгҖӮжңҚеҠЎз«Ҝи§ЈжһҗеҚҸи®®жӢҝеҲ°еҠ еҜҶж–№ејҸеҗҺпјҢеҶҚеҠЁжҖҒи§ЈеҜҶжҠҘж–Ү
