# Pulsar
FluxMQ提供了Pulsar的接入功能，可以将消息发送到Pulsar中。
## Pulsar配置
![img_2.png](../../../../assets/images/gzyq/source/img_2.png)

| 配置项               | 说明                                   |
|-------------------|--------------------------------------|
| Pulsar地址          | Pulsar的地址，比如：pulsar://localhost:6650 |
| ioThreads         | IO线程数、可选                             |
| connectionTimeOut | 连接超时时间、可选                 3          |
