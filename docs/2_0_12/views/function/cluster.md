# 集群功能

## 初始化集群

> 三节点集群

| 节点           | 
|--------------|
| 192.168.1.10 |
| 192.168.1.11 |
| 192.168.1.12 |

### 节点配置
```yaml
cluster: # 集群配置
  localAddress: "192.168.1.10" # 配置启动节点的IP
  addresses: [ "192.168.1.10","192.168.1.11","192.168.1.12"]
```

## 新增节点
| 节点           | 
|--------------|
| 192.168.1.13 |

### 节点配置
> 只需要填写集群任意IP，即可互相发现

```yaml
cluster: # 集群配置
  localAddress: "192.168.1.13" # 配置启动节点的IP
  addresses: [ "192.168.1.10","192.168.1.1","192.168.1.12","192.168.1.13"]
```
