# 支持设置redis脚本

目前支持的指令如下：

指令 描述
set key value 设置key-value
del key 删除key
expire key 设置key TTL时间，单位秒
incr key 自增计数
decr key 自减计数
hset key field value 添加Hash值
hdel key field 删除hash值

支持设置多指令,设置规则如下:

 指令1 && 指令2 && 指令3
